Задача 2.1: Вычисление квадратного корня методом Ньютона

Описание: Реализовать вычисление квадратного корня с заданной точностью ε с использованием итерационной формулы Ньютона. Для каждого входного числа x вычислить √x, пока разность между последовательными приближениями не станет меньше ε.


Задача 2.2.1: Схема Горнера

Описание: Вычислить значение многочлена в точке x по схеме Горнера. Коэффициенты многочлена задаются в порядке от старшего к младшему.


Задача 2.2.2: Схема Горнера + производная

Описание: Вычислить значение многочлена и его производной в точке x по схеме Горнера.


Задача 2.2.3: Многочлен, производная, интеграл

Описание: Вычислить значение многочлена, его производной и интеграла в точке x. В начале вводится степень многочлена n.


Задача 2.3: Числа Фибоначчи

Описание: Для каждого входного целого числа i вычислить i-е число Фибоначчи двумя способами: итеративно и рекурсивно.


Задача 2.4: Парсер вещественных чисел

Описание: Реализовать функцию str2double для преобразования строки в вещественное число без использования стандартных функций.


Задача 2.5: Список слов

Описание: Построить список из слов строки, затем исключить все слова, совпадающие с последним словом (кроме самого последнего).


Задача 2.6: Дерево поиска

Описание: Реализовать бинарное дерево поиска с операциями добавления (+), удаления (-) и проверки наличия (?).


Задача 2.7: Расширенный калькулятор

Описание: Реализовать калькулятор выражений с операциями +, -, *, /, ^ и скобками методом рекурсивного спуска.
