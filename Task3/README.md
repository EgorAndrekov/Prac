*Описание программы*

Программа читает входные данные из стандартного ввода (stdin), разбивает текст на слова (обычные и специальные), выводит последовательность слов в исходном порядке, а затем в лексикографическом порядке.

*Функциональность*

Чтение данных: Программа читает данные построчно до достижения конца файла (EOF)

Разбор слов: Автоматически различает обычные слова и специальные символы

Обычные слова: Состоят из букв, цифр и символов: $, ', `, /, ,

Специальные слова: |, ||, &, &&, ;, >, >>, <, (, )

Вывод результатов: Для каждой строки выводится:

- Количество слов

- Слова в исходном порядке

- Слова в лексикографическом порядке
